## Commands Now Available

```bash
# Terminal commands
maximus                    # Start chat session
maximus --new              # Fresh session
maximus --session <id>     # Resume session
maximus --model <name>     # Select model
maximus doctor             # Diagnostics

# In-session commands
/mcp add <url>             # Add MCP server
/mcp list                  # List MCP servers
/mcp discover              # Auto-discover servers
discover <query>           # Search packages
sandbox run <code>         # Execute in sandbox
```
